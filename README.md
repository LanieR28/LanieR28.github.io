# LanieR28.github.io

This repository is the source for the personal website of `LanieR`.

## Deploy with GitHub Pages

1. Create a GitHub repository named `LanieR28.github.io`
2. Push the files in this directory to the `main` branch
3. In GitHub repository settings, open `Pages`
4. Set deployment source to `Deploy from a branch`
5. Select `main` and `/ (root)`

The site will be published at:

- `https://lanier28.github.io`

## Included content

- Personal homepage
- Featured CHUNITHM bot system project section
- Sample rendered assets copied from the current local project
