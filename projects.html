アーカイブ

制作物とシステムの記録

個人サイトから見える制作物を、もう少しだけプロジェクト寄りに整理したページです。 現時点では CHUNITHM Bot System が中心ですが、今後ここに少しずつ増やしていく前提のアーカイブとして使います。

主力プロジェクト

CHUNITHM Bot System

QQ 上で操作する Bot を中心に、Best 30、譜面プレビュー、楽曲情報、 レベル一覧、許容値分析、アカウント管理、公式サイト同期を一体化した個人制作システムです。 使う人にとって実用的であることと、外から見たときに成果物として説明しやすいことの両方を意識しています。

入力 QQ コマンド / 公式サイトデータ / 手動登録
出力 画像カード / 譜面表示 / 集計データ
中核 Bot Bridge / Renderers / Sync Scripts
重視点 見た目・使いやすさ・データ整理
  • Best 30、AJ、AJC、PC30 の画像化表示
  • 譜面プレビューと楽曲情報カードの高画質出力
  • 別名検索、難易度補完、アカウント作成とバインド
  • CHUNITHM-NET 公式サイトとの同期フロー

周辺機能

譜面プレビュー表示フロー

譜面データの取得、キャッシュ、変換、送信の流れそのものも一つのミニシステムとして扱っています。 画質維持と送信制約のバランスを取りながら、実際に使えるプレビュー体験を設計しています。
